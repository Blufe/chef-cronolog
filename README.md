Description
===========

Installs the cronolog package.

Requirements
============

## Platform:

* Debian/Ubuntu
* Redhat/CentOS
* Other platforms untested

Usage
=====

Include ```recipe[cronolog]``` on runlist for the node/role you want to install cronolog on.
